# dofi
rofi UI for todo.txt

# Screenshot
![Screenshot]
(images/image.jpg)

# Features:

* Show your tasks (duh!)
* Basic tasks management (mark done, delete, set priority)
* Associate URLs with tasks (add, delete, open)
* Associate notes with tasks (add, edit, delete)
