# dofi
rofi UI for managing simple todos

# Screenshot
![Screenshot]
(images/image.jpg)

# Features:

* Show your tasks (duh!)
* Basic tasks management (mark done, delete, set priority)
* Associate URLs with tasks (add, delete, open)

# Dependencies:

* gawk
* grep
* sed
* rofi
* linux-utils (column)

# Installation

1. Copy dofi to $PATH
2. Copy dofi.config to $HOME/.config/dofi/config
3. Run dofi
